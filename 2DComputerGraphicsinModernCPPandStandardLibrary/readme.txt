clang++ or g++ to compile.
Compiles with C++20 and above.
Only tested on macOS 13.6.3 Ventura and Windows 10 with msys2.

Build and run all:
./build_and_run_all.sh

(if it doesn't work: chmod a+x build_and_run_all.sh
And try again)

All output from binaries will be saved into 'output' directory.
All binaries will be be moved to 'builds' directory.
All generated .ppm images will be moved to 'ppm' directory.
